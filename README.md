# Бэк для LexaOS на Nest.js + Mongo + Redis

[Основной репозиторий](https://github.com/Rainatnight/lexaos)

Что кэшируется: папки пользователя на главной странице (getFolders)

Почему: данные не часто меняются, но запрашиваются часто, поэтому кэш снижает нагрузку на MongoDB и ускоряет отклик

TTL: кэш обновляется каждые 2 минуты, или сразу после изменения данных (создание, удаление, переименование папки)
